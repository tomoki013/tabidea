# What Is Tabidea?

更新日: 2026-03-03  
Last updated: 2026-03-03

## 1. サービス概要 / Service Overview

Tabidea（タビデア）は、AIを使って「旅行先を決めるところ」から「旅程の詳細化」「共有・再編集」までを一つの体験で提供する旅行プランニングサービスです。  
Tabidea is an AI travel planning service that covers destination discovery, itinerary generation, refinement, and sharing in one flow.

### 何ができるか / What you can do

- 条件入力から数秒で旅行プランのアウトラインを作成
- 日別の詳細行程（時間帯、スポット、移動）を生成
- 生成後にチャットで「もっとゆっくり」「予算を抑えたい」など調整
- プランを保存して、公開リンク（しおり）として共有
- 渡航情報（安全、気候、ビザ、マナーなど）を取得
- 航空券/ホテル候補の検索結果をプランに反映（対象プラン）
- 旅の振り返り（Journal）や公開コンテンツ連携（Blog）

## 2. 誰のためのサービスか / Who Is It For

### 主な対象ユーザー / Primary users

- 旅行は好きだが、行程を細かく作る時間がない人
- どこに行くか迷っていて、候補提案から始めたい人
- 家族旅行・友人旅行・一人旅など、条件が複雑な人
- 旅行メモやしおりを公開して共有したい人

### 想定シーン / Typical use cases

1. 週末旅行の短期プラン作成
2. 複数都市周遊のルート設計
3. テーマ旅行（グルメ、絶景、聖地巡礼）
4. 既存プランの改善（再生成・再調整）

## 3. 使い方（利用者目線） / User Journey

### Step 1: 条件を入力

ホーム画面のプランナーで、以下を選択します。

- 目的地（既に決まっている or 未定）
- 日程
- 同行者
- テーマ
- 予算
- ペース
- 行きたい場所、移動希望、固定予定（予約済み交通・宿など）

### Step 2: AIがアウトラインを作成

AIが、入力条件に加えて関連記事コンテキストを参照し、まず全体構成（何日目にどこを回るか）を生成します。

### Step 3: 日別詳細を生成

アウトラインに基づいて、日ごとの行程（活動・移動・説明・参考情報）を生成します。

### Step 4: 必要に応じて再調整

チャットで要望を伝え、行程を調整できます。

例:
- 「移動を減らして」
- 「子ども向けスポットを増やして」
- 「予算を1段階下げて」

### Step 5: 保存・共有・公開

- マイプランに保存
- 公開URLで共有（Shiori）
- 必要に応じてBlogに埋め込み

## 4. 主な機能 / Core Features

### 4.1 AIプラン生成

- 2段階生成: アウトライン -> 詳細
- 旅行テーマ、予算、ペース、固定予定を反映
- 複数都市の周遊にも対応

### 4.2 渡航情報

カテゴリ例:
- 基本情報（通貨、言語、時差）
- 安全・医療
- 気候・服装
- ビザ、マナー、交通、通信、喫煙/飲酒など

### 4.3 再計画（Replan）

状況変化（時間不足、体力、混雑など）に応じた代替案を評価して提示します。

### 4.4 共有・公開

- 共有リンクで公開表示
- 公開状態の制御（private/unlisted/public）
- 閲覧向けのしおり表示

### 4.5 Blog連携

- ブログ記事にしおり埋め込み
- 旅行体験の発信と再利用

## 5. 料金・利用制限（実装ベース） / Pricing and Limits

実装上は以下のユーザー種別で制限が定義されています。

- anonymous
- free
- pro
- premium
- admin

代表的な制限:

- プラン生成回数
- 渡航情報取得回数
- 渡航情報カテゴリの閲覧範囲
- 地図機能・Places詳細・外部候補数

詳細な制限値は運用時に変更される可能性があるため、最新値はアプリ画面と`src/lib/limits/config.ts`を参照してください。

## 6. 情報の扱いと品質 / Data Quality and Trust

- 渡航情報は複数ソースを統合して返します
- 情報の鮮度や信頼性を評価する実装があります
- 公的情報に基づくカテゴリでも、最終判断は公式情報で再確認してください

## 7. よくある質問（簡易） / Quick FAQ

### Q. 目的地が決まっていなくても使える？
A. 使えます。地域・テーマ・雰囲気から候補提案を含むプランを生成できます。

### Q. 生成したプランは後で編集できる？
A. 可能です。再生成・チャット調整・公開設定変更ができます。

### Q. 無料でどこまで使える？
A. 基本機能は無料で利用可能です。回数や一部高度機能はプラン種別で制限されます。

### Q. 情報は100%正確？
A. AI生成要素を含むため、重要事項（ビザ・安全情報・規制）は必ず公式情報で確認してください。
